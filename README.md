🎮 Simon Game
A browser-based implementation of the classic Simon memory game, built using HTML, CSS, JavaScript, and jQuery. Test your memory by repeating an increasingly complex sequence of colors.

Live Demo
Play the Game Here: https://asnemitha.github.io/Simon-Game/
Overview
This project recreates the classic Simon Game where players must remember and repeat a randomly generated sequence of colors. With each level, the sequence grows longer, increasing the difficulty.

Features
-Four interactive color buttons
-Sound feedback for each button
-Visual animations (fade and press effects)
-Level progression system
-Game-over detection with restart option
-Start game using keyboard input

Tech Stack
1.HTML5 – Structure
2.CSS3 – Styling and layout
3.JavaScript (ES6) – Core logic
4.jQuery (v3.7.1) – DOM manipulation & effects

Game Logic
1.Press any key to start the game
2.A random color flashes
3.Repeat the sequence by clicking buttons
4.Each correct round adds a new step
5.A wrong click triggers:
    -Game over sound
    -Screen flash effect
    -Restart prompt
